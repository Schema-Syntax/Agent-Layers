# Agent-Layers

An interactive reference tool for learning the concepts, patterns, and infrastructure of agentic AI — organized by abstraction layer so that position on the page actually means something.

## What it is

A self-contained HTML file. No dependencies, no build step, no internet connection required. Open it in any browser and it works.

The 68 tiles are arranged across eight layers, from foundational concepts at the top to real-world applications at the bottom. Within each layer, tiles cluster by domain. Vendor and tool entries are visually distinguished with a dashed border — they're reference entries, not conceptual peers.

## How to use it

Click any tile to open an expanded definition, including context about why it sits where it does, related concepts you can jump to directly, and keyword tags for orientation. The search bar filters across tile names, definitions, categories, and tags. The category buttons along the top let you isolate a single domain. Mark tiles as studied as you go — progress persists in your browser.

Keyboard shortcuts: `←` `→` to navigate between tiles, `S` to mark studied, `Esc` to close.

## Why the layers matter

The original concept this was built from used a periodic table. The goal here was to encode meaningful positional logic. The eight layers represent genuine abstraction levels:

1. **Foundations** — what AI is at the conceptual level
2. **Core Capabilities** — what an agent can do as primitive operations
3. **Memory Concepts** — how agents think about retaining information
4. **Memory Implementation** — where and how information is stored and retrieved
5. **Patterns & Architecture** — how agents are structured and orchestrated
6. **Evaluation & Safety** — how agents are governed and validated
7. **Infrastructure & Deployment** — how agents are built and run
8. **Applications** — what agents accomplish in the world
